## [1.0.28] - 2026-07-21

**PR:** #9  

**새 기능**
- 앱 시작 화면이 새로 추가되었습니다
- 아이 호칭과 도움 목표를 설정하는 과정이 추가되었습니다
- 하루를 함께할 친구 캐릭터를 고를 수 있습니다
- 보호자 전용 비밀암호를 설정할 수 있습니다
- 일과를 말하듯 적으면 행동 카드로 만들어 줍니다
- 카드를 만들기 전 필요한 준비물을 되물어 봅니다
- 만들어진 카드를 확인하고 수정한 뒤 시작할 수 있습니다
- 관리자 화면에서 회원과 일과 내역을 조회할 수 있습니다
- 관리자 화면에서 일과 단계별 그림을 확인할 수 있습니다

**개선**
- 개인정보가 지워지는 과정을 눈으로 확인할 수 있습니다
- 관리자 화면 디자인이 전반적으로 정돈되었습니다
- 다양한 화면 크기에서 화면 배치가 자연스럽게 맞춰집니다

**버그 수정**
- 아이 호칭을 입력하지 않으면 안내 문구가 어색하게 보이던 문제 해결
- 일부 화면에서 그림이 잘려 보이던 문제 해결
- 관리자 화면에서 오류 발생 시 안내가 표시되지 않던 문제 해결

## [1.0.27] - 2026-07-21

**PR:** #6  

**새 기능**
- 보호자가 아이의 일과를 직접 입력할 수 있습니다
- 입력한 내용에서 개인정보를 자동으로 가려줍니다
- 전송 전과 후의 문장을 나란히 확인할 수 있습니다
- 필요한 준비물을 되묻는 안내가 추가되었습니다
- 생성된 행동 카드를 검토하고 승인할 수 있습니다
- 앱 아이콘이 새로 적용되었습니다

**버그 수정**
- 앱 설치 파일이 만들어지지 않던 문제를 해결했습니다

**문서**
- 프로젝트 소개와 사용 안내를 새로 정리했습니다
