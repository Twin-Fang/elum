## [1.0.35] - 2026-07-21

**PR:** #22  

**새 기능**
- 보호자 홈 화면이 새 디자인으로 바뀌었습니다
- 상황별 추천 일과를 좌우로 넘겨 고를 수 있습니다
- 추천 일과를 고르면 내용이 미리 채워진 채 시작됩니다
- 만들어 둔 일과를 홈에서 바로 확인할 수 있습니다
- 아이 이름을 입력하면 자동으로 시작됩니다
- 같은 이름을 다시 입력하면 이전에 만든 일과가 그대로 보입니다

**개선**
- 일과와 카드가 서버에 저장되어 앱을 껐다 켜도 남아 있습니다
- 오래 사용해도 로그인이 풀리지 않습니다
- 연결이 불안정해도 홈 화면이 비어 보이지 않습니다

**버그 수정**
- 일부 환경에서 앱 실행 시 오류가 나던 문제를 해결했습니다

## [1.0.32] - 2026-07-21

**PR:** #15  

**새 기능**
- 아이를 부를 호칭을 입력하는 화면이 추가되었습니다
- 도와주고 싶은 순간을 여러 개 고를 수 있습니다
- 하루를 함께할 친구를 고를 수 있습니다
- 보호자 전용 비밀번호를 만들 수 있습니다
- 설정을 마치면 안내 화면이 나타납니다

**개선**
- 비밀번호 입력에 기기 기본 숫자 키패드를 사용합니다
- 선택한 항목이 더 뚜렷하게 구분되어 보입니다
- 화면 구성과 여백을 디자인에 맞게 다듬었습니다
- 비밀번호를 다 입력한 뒤 직접 확인하고 넘어갈 수 있습니다

**버그 수정**
- 선택 항목의 강조 색상이 잘못 표시되던 문제를 해결했습니다

## [1.0.28] - 2026-07-21

**PR:** #9  

**새 기능**
- 앱 시작 화면이 새로 추가되었습니다
- 아이 호칭과 도움 목표를 설정하는 과정이 추가되었습니다
- 하루를 함께할 친구 캐릭터를 고를 수 있습니다
- 보호자 전용 비밀암호를 설정할 수 있습니다
- 일과를 말하듯 적으면 행동 카드로 만들어 줍니다
- 카드를 만들기 전 필요한 준비물을 되물어 봅니다
- 만들어진 카드를 확인하고 수정한 뒤 시작할 수 있습니다
- 관리자 화면에서 회원과 일과 내역을 조회할 수 있습니다
- 관리자 화면에서 일과 단계별 그림을 확인할 수 있습니다

**개선**
- 개인정보가 지워지는 과정을 눈으로 확인할 수 있습니다
- 관리자 화면 디자인이 전반적으로 정돈되었습니다
- 다양한 화면 크기에서 화면 배치가 자연스럽게 맞춰집니다

**버그 수정**
- 아이 호칭을 입력하지 않으면 안내 문구가 어색하게 보이던 문제 해결
- 일부 화면에서 그림이 잘려 보이던 문제 해결
- 관리자 화면에서 오류 발생 시 안내가 표시되지 않던 문제 해결
