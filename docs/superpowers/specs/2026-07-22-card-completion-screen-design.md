# 카드 생성 완료 화면 설계

**작성일**: 2026-07-22  
**Figma 노드**: 425-4199 (보호자_새로운 일과 만들기_완료)

## 개요

보호자가 새로운 일과를 입력한 후 AI가 행동 카드 생성을 완료한 상태를 표시하는 화면입니다.  
Image #6 (90% 진행)에서 Image #7 (100% 완료)로 전환되며, 1.5초 표시 후 보호자 확인 UI로 이동합니다.

## UI 구성

### 1. 배경 (기존 코드 재사용)
- 초록(#78FFB0) → 파란색(#0099FF) 그라데이션
- blur 효과 포함
- 기존 splash_screen 또는 유사 화면의 배경 컴포넌트 재활용

### 2. 별 아이콘 (기존 코드 재사용)
- 중앙 상단 배치
- 기존 디자인 시스템의 별 아이콘 컴포넌트 사용

### 3. 제목 텍스트 (신규 구현)
- 텍스트: "내용 정리가 모두 완료됐어요"
- 타입: 헤더(H1)
- 색상: 다크 네이비(#1a1a1a)
- 폰트: 프로젝트 기본 폰트
- 정렬: 중앙
- 위치: 별 아이콘 하단, 화면 중앙

### 4. 진행도 텍스트 (신규 구현)
- 텍스트: "100% 완료!"
- 타입: 서브텍스트
- 색상: 라이트 그레이(#A0A0A0)
- 폰트: 프로젝트 기본 폰트 (작은 크기)
- 정렬: 중앙
- 위치: 제목 텍스트 하단

### 5. DLP 배지 (신규 구현)
- 위치: 화면 하단
- 텍스트: "secured by ELUM AI DLP"
- 기존 `SecuredByDlpBadge` 컴포넌트 재사용 (있는 경우)
