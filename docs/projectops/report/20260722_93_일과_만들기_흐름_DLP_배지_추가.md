# 일과 만들기 흐름 화면에 DLP 보안 배지 추가

## 개요

Figma에 이미 디자인되어 있던 "secured by ELUM AI DLP" 신뢰 배지(자물쇠 아이콘 + 문구)를 시작 화면과 일과 만들기 흐름 6개 화면, 총 7개 화면 하단에 실제로 구현했다. 화면마다 좌표를 반복해 넣지 않고 공통 위젯 하나로 만들어 재사용했다.

## 변경 사항

### 신규 에셋 · 위젯

- `client/assets/images/icon_dlp_lock.svg`: Figma Component 7(418:4049)에서 다운로드한 자물쇠 아이콘 원본
- `client/lib/core/widgets/secured_by_dlp_badge.dart`: 자물쇠 아이콘 + "secured by ELUM AI DLP" 문구를 그리는 공통 위젯 `SecuredByDlpBadge`. 텍스트는 Pretendard SemiBold 14, 색상 `#74757D`
- `client/lib/core/assets/app_assets.dart`: `iconDlpLock` 경로 상수 추가

### 화면 반영

- `client/lib/features/guardian/presentation/widgets/routine_flow_scaffold.dart`: 공통 스캐폴드 하단에 배지를 추가. 이 스캐폴드를 쓰는 로딩(prepare/generate) 2개, 추가질문 2개, 카드확인 화면까지 총 5개 화면에 자동 반영됨
- `client/lib/features/onboarding/presentation/splash_screen.dart`: 시작 화면(238:1808) 하단에 Figma 좌표(x=0, y=756, 폭 393 기준 중앙 정렬)로 개별 추가
- `client/lib/features/guardian/presentation/routine_input_screen.dart`: 자체 `Scaffold`를 쓰는 화면이라 공통 스캐폴드 대상이 아니어서 기존 `_PrivacyNote` 문구 아래에 개별 추가

## 주요 구현 내용

- 7개 화면이 모두 화면 하단 거의 동일한 위치(Figma 기준 x≈107~127, y≈756~783)에 배지를 두고 있어, `RoutineFlowScaffold`를 쓰는 화면군은 스캐폴드 레벨에서 한 번만 넣어 5개 화면을 동시에 커버했다. 스캐폴드를 쓰지 않는 시작 화면과 입력 화면만 개별 추가했다.
- 자물쇠 아이콘은 도형을 코드로 그리지 않고 Figma 원본 SVG를 그대로 받아 사용했다 (그라데이션·복잡한 path가 섞인 아이콘이라 재현 대신 원본 사용).
- 색상 `#74757D`는 기존 `AppColors` 팔레트에 없는 값이라 위젯 내부에 지역 상수로 정의했다.

## 주의사항

- `#74757D`는 이 배지 전용으로만 쓰인다. 다른 화면에서 같은 회색 텍스트가 필요해지면 `AppColors`에 정식 토큰으로 승격하는 것을 검토한다.
- 커밋: `336b694`
